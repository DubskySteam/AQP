Example 4: Use actual date / time

To show the actual time to the user use a calculated dataset. If user selects another date / time, dataset is updated and date / time stays fixed.
Important note: The time to automatic reload the component should not set to a value below 5 seconds, to give users enough time to change the date / time, before it refreshs.


            <div id="selectdatetime_example4" swa="Selectdatetime FROM ../../data/selectdatetime/exampleEmptyDatetimes.json"></div>
                    

            selectdatetime_example4_options = {
                actualTimeForEmpty: true,
                reloadInterval: 5
            };

});

Example 6: Listening to actual time update

You can listen for a timechange when the component is in reload mode, by listening to the swac_REQUESTOR.ID_reloaded event.


document.addEventListener('swac_selectdatetime_example4_reloaded', function (evt) {
    let requestor = evt.detail;
    console.log('The actual time is: ');
    console.log(requestor.querySelector('.swac_repeatedForSet .swac_selectdatetime_time').getAttribute('value'));
});
