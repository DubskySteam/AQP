Dieses Verzeichnis enthält Dateien für den Test von Datei up und downloads.

Folgende Dateien sind einzeln verwendbar:
Box.glb - glTF binary (binärdaten mit eingebetteten jsondaten, json ist maschinenlesbar)
Box-embedded.gltf - glTF embedded (json mit eingebetteten binärdaten)
CesiumMilkTruck.glb - glTF binary (etwas komplexeres Modell)
house.obj - obj 3D Datei sowohl einzeln hochladbar und im Frontend darstellbar, als auch in Kombination mit der House.mtl

Nur zusammen verwendbare Dateien: (Diese Dateien müssen immer paarweise hochgeladen werdden)
House.obj - obj 3D Datei
House.mtl - Materialdatei zur House.obj

Box.gltf - gltf json Datei
box0.bin - binärdatei mit Vertices für die Box.gltf

House.gltf - gltf json Datei (kompexeres Modell)
buffer_combined.bin - binärdatei mit Vertices für die House.gltf
