Data Example 14: Calculated default values

You can use calculated default values. Write an equation with names of existing attributes in % (as placeholder) and a javascript supported equation.


var data_example14_1_options = {
  attributeDefaults: new Map([['../../data/exampledata_list.json',{multiply: '%doubleval% + %intval%'}]])
};
