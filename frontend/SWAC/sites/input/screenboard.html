Example 3: Special button functions

You can give all buttons special functions to be executed on tipp. 

<div swa="Screenboard"></div>
        

var screenboard_options = {};
screenboard_options.specialButtons = [];
screenboard_options.specialButtons[0] = {
    key: 'icon: sign-in',
    func: function(evt) {
        alert('Special button pressed = custom function executed!');
    }
};
