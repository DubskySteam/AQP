Support for legacy browsers

SWAC 27.02.2022 introduces support for legacy browsers. Support is limited to view of data from variables, but supports browsers that miss import support.

Activate support

To activate legacy support add the legacy script to the imported script files in page header.


<script src="/SWAC/swac/legacybrowser.js"></script>

Special areas for legacy browsers

To show special content for legacy browsers just add the classes swac_dontdisplay swac_legacy 


<div class="swac_dontdisplay swac_legacy">
    This content is only viewd to legacy browsers
</div>
