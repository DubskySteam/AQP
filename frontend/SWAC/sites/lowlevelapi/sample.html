The sample component is a template component as startpoint for your own components. It shows the possible esiest structure and how to define options, use templates and activate plugins.
Also this page is a sample for a documentation page. You can find it at sites/lowlevelapi/sample.html.
The js/lowlevelapi/sample.js belonging to this site gives you an example on how to configure a component.

Examples

Example 1: Use sample component

A simple example on how to use a component.


<div id="example1" swa="Sample"></div>

Example 2: Use sample with plugin

This is an example of the sample component with enabled plugin, that contains a subcomponent.


<div id="example2" swa="Sample"></div>
        

var example2_options = {
    plugins: new Map()
};
example2_options.plugins.set('SamplePlugin', {
    id: 'SamplePlugin',
    active: true
});
