Example 1: Simple labeling

Add a simple labeling feature to your datasets.
Note that adding a label by click on the labels name will not work in this example, because the example file is not writeable.


var labeling_example1_options = {
    showWhenNoData: true,
    labelidAttr: 'label_id',
    labeledidAttr: 'oo_id',
    labelSource: {
        fromName: '../../data/input/labeling/label_labels.json'
    },
    showNoDataInfo: false
};
